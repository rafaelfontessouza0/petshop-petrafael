AJUDA - SISTEMA WEB PETRAFAEL

1. index.html (Início)
   - Página inicial do sistema web do petshop PetRafael.
   - Exibe o cabeçalho com o nome do petshop, slogan e menu de navegação para as demais páginas (Produtos, Serviços e Contato).
   - Apresenta um texto de boas-vindas explicando que o PetRafael vende produtos e oferece serviços de banho e tosa.
   - Contém um pequeno resumo das três categorias de produtos (acessórios, rações não perecíveis, higiene e limpeza), com um exemplo de produto e valor para cada categoria.
   - Destaca os serviços de banho e tosa, convidando o usuário a acessar a página de serviços para mais detalhes.
   - Possui rodapé com informações de direitos autorais, nome do autor, curso, disciplina e dados de contato.

2. produtos.html (Produtos)
   - Página dedicada à visualização dos produtos do petshop.
   - Organiza os produtos em três seções principais: Acessórios, Rações não perecíveis e Higiene e limpeza.
   - Em cada seção há pelo menos dois produtos com nome, pequena descrição, imagem ilustrativa (link externo) e valor.
   - Permite ao usuário visualizar, de forma simples, exemplos de itens que o PetRafael oferece em cada categoria.
   - Repete o cabeçalho e o rodapé padrão do site, facilitando a navegação entre as páginas.

3. servicos.html (Serviços)
   - Página que descreve os serviços oferecidos pelo PetRafael.
   - Apresenta os serviços de banho e tosa em duas modalidades: com tele-busca e sem tele-busca.
   - Para cada tipo de serviço são exibidas descrição e valores, deixando claro o que está incluído no atendimento.
   - Explica que a opção de tele-busca permite que o pet seja buscado e devolvido na residência do cliente, reforçando o requisito do projeto.
   - Mantém o cabeçalho e o rodapé padrão, com navegação para as demais páginas do sistema.

4. contato.html (Contato)
   - Página com os dados de contato do PetRafael.
   - Exibe endereço (fictício), e-mail, telefone e horário de funcionamento do petshop.
   - Contém um formulário estático em HTML, sem funcionalidade de envio, com campos de nome, e-mail, tipo de interesse e mensagem, além de um botão de envio.
   - Esse formulário serve apenas para demonstrar a estrutura de coleta de dados de contato no contexto da disciplina.
   - Repete o cabeçalho e o rodapé padrão do site.

Observações gerais:
   - Todo o sistema foi desenvolvido apenas com HTML, conforme solicitado para a Fase 1 do projeto, sem uso de CSS, Bootstrap ou JavaScript.
   - As páginas utilizam cabeçalho e rodapé padronizados, contribuindo para a navegação entre as diferentes partes do site.
   - Os arquivos objetivos-metas.txt e README-ajuda.txt servem de apoio para a documentação do projeto e para a elaboração do PDF a ser entregue na disciplina.
